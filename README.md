🎬 **Film Sitesi Tasarımı**

Visual Studio ortamında kendimi geliştirmek amacıyla oluşturduğum bir deneme projesidir. Bu projede temel web tasarım yapısını deneyimlemek, sayfa düzeni oluşturmak ve kullanıcı arayüzü tasarımını geliştirmek üzerine çalıştım.

Projede; film temalı bir web sitesi arayüzü tasarlayarak sayfalar arası yapı, içerik düzeni ve temel tasarım prensiplerini uygulamaya çalıştım.

💡 Amaç:

* Visual Studio kullanım pratiğimi artırmak
* Web arayüzü tasarımını geliştirmek
* Proje mantığını ve sayfa yapısını daha iyi anlamak

Bu proje tamamen öğrenme ve pratik amaçlı hazırlanmıştır.
